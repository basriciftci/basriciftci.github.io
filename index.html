Next Word Prediction and
Auto-completion of Turkish Words using Char-RNN

Istanbul Sehir University, ECE 520 Final Project
Basri Çiftçi, Ömer Faruk Turgut

Machine Learning itself is a subfield of Artificial Ingellince. Our modern world is witnessing new technologies every day, which are mostly to make our daily actions and works much more easier. Natural Language Processing is one of the those fields in Machine Learning which is in great demand nowadays. There are numerous application like online translation, text summarization in NLP studies. Especially after new techniques and algorithms have been developed in the field of deep learning, big companies have made some major improvements in this field as well. But the vast majority of studies are in widely spoken languages such as English. For languages like Turkish we have seen the lack of studies. So with my friend, we tried to develop an application for Turkish language.

The main motivation and purpose behind this project is to have better features for a smartphone keyboard, especially for Turkish language. There are some fundamental features of a smartphone keyboard, like auto-correction, spell-checking, auto-completion and next-word-prediction when you type on texts. In this study we worked on only next-word-prediction and auto-completion of a Turkish word.
